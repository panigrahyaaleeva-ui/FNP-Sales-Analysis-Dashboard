FNP Sales Performance Dashboard

Project Overview
This project features an interactive dashboard designed to analyze sales performance for Ferns N Petals (FNP). It transforms raw transactional data into actionable business intelligence to optimize inventory and marketing strategies.

Key Insights

Key Performance Indicators (KPIs)
>Total Revenue: ₹35,20,984.00 across 1,000 total orders.
>Average Order Value (AOV): ₹3,520.98.
>Operational Efficiency: Average order-to-delivery time of 5.53 days.

Strategic Visualizations
>Temporal Trends: Revenue spikes in February, March, and August; peak ordering occurs between 6:00 PM and 10:00 PM.
>Product Performance: "Colors" and "Soft Toys" are leading categories; the "Magnam Set" is the top-selling product.
>Occasion-Based Demand: Sales are heavily driven by Raksha Bandhan (18%) and Anniversaries (19%).
>Geographic Distribution: Order volume is highest in Imphal, Kavali, and Dhanbad.


Tools Used
>Data Visualization: [Power BI / Excel / Tableau]
>Data Cleaning: Power Query
>Analysis: DAX / Formulas
